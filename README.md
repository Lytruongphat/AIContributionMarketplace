# Title
AI Contribution Marketplace

# Description
AI models rely heavily on human-annotated data, yet the contributors who label this data often face opaque payment systems, delayed compensation, or exploitation by centralized intermediaries. I built this project to create a decentralized, transparent, and automated DataFi marketplace. By leveraging Stellar's low-cost and high-speed blockchain, users are paid instantly and fairly for their AI data contributions without the need for middlemen.

# Features
* **Decentralized Data Uploading:** Users can upload tasks (datasets/images) paired with a reward pool in XLM.
* **Majority Consensus Moderation:** A decentralized verification system where tasks are automatically approved only when a specific threshold of workers (e.g., 3 people) agree on the same label, acting as a smart-contract-based referee.
* **Instant Micropayments:** Soroban smart contracts automatically and evenly distribute XLM rewards to the correct annotators the exact moment consensus is reached.
* **Anti-Spam Mechanism:** Built-in validation prevents double-voting and sybil attacks to ensure high-quality AI data.
